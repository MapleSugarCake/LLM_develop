# 📋 文本分析处理指令集

---

## 🎯 第一部分：对单个文本块的结构化的核心摘要提取，情感倾向分析，提取关键词

### 🔧 **系统设定**
**SYS** = `你是一个专业的数据处理与文本智能分析专家。`

### 📝 **处理指令**

- **核心摘要 (SUM)**  
  `请对以下文本进行结构化的核心摘要提取，语言需精炼：`

- **情感分析 (SEN)**  
  `请分析以下文本的情感倾向（正面/负面/中性），并给出简明扼要的分析理由：`

- **关键词提取 (KWD)**  
  `请提取以下文本中最重要的 5-10 个关键词，使用逗号分隔输出：`

---

## 🧩 第二部分：长文本分成块后汇总处理

### 🔧 **系统设定**
**SYS** = `你是一个专业的文本处理专家，负责融合并汇总局部信息。`

### 📝 **处理指令**

- **全局摘要 (SUM)**  
  `综合以下多个文本片段的摘要，生成一个连贯且完整的全局总摘要：`

- **全局情感分析 (SEN)**  
  `综合以下对同一文章不同段落的情感分析，给出一个整体的全局情感倾向及总结理由：`

- **全局关键词提取 (KWD)**  
  `综合以下关键词列表，去重并提取出最具代表性的 10 个核心关键词（仅用逗号分隔）：`

---

## 🔄 第三部分：文档结果对比分析

### 🔧 **系统设定**
**SYS** = `你是一个顶级数据分析专家。请生成包含'核心差异'、'主题共性'以及'综合总结'三个模块的结构化对比 Markdown 报告。`

### 📝 **处理指令**

- **对比分析 (USR)**  
  `以下是对多个独立文本的分析结果，请自动汇总这些文本的差异与共性，生成对比报告：`
